## Keylogger

This software must only be used on systems you "own" or have explicit permission to monitor.

Using a **keylogger** without user consent may be **illegal** and unethical, and **could** violate privacy laws and cybersecurity regulations in many countries.

## "Do NOT" use this program to:

- Spy on users without consent

- Steal passwords or private information

- Monitor devices you do not own

## Configuration
1. Install venv **(recommend)**:
   ```bash
   python -m .venv .venv
   .venv\Scripts\Activate
   ```
2. Install python dependens :
    ```bash
    pip isntall requirements.txt
    ```
3. Execute:
   ```bash
   py app.py
   ```
